This webpage was designed as part of a software engineering module. The aim of this project is to create a program which allows the user to monitor radicalised individuals by analysing Twitter feeds

Users have the option of searching by keywords or directly by username. Our algorithm uses a database of trigger words and artificial intelligence to determine the threat level of individual tweets or users

Disclaimer:

The results on this website do not relfect our personal views. This program is a concept and therefore is not 100% accurate at determining terrorist threats. This should not be used as a standalone tool but rather as a tool to aid investigation.
